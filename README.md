# ☕ CafeCrust

CafeCrust is a modern and responsive café website built using HTML, CSS, Bootstrap, and JavaScript. It provides an interactive online menu where users can browse different food categories, search for items, add products to a shopping bucket, and place orders through a checkout system.

## 🌐 Live Demo

https://zaid-2006.github.io/CafeCrust/

## ✨ Features

- 🎨 Modern and responsive UI
- ☕ Hot Coffee section
- 🥤 Cold Coffee section
- 🍕 Pizza section
- 🍔 Burger section
- 🧁 Bakery & Dessert section
- 🔍 Real-time search functionality
- 🛒 Shopping bucket/cart
- ➕ Increase & decrease product quantity
- 💰 Automatic total price calculation
- 📍 Checkout form with delivery details
- 💳 Multiple payment options
  - Cash on Delivery
  - UPI
  - Credit/Debit Card
- 📱 Mobile-friendly design
- 🚀 Smooth scrolling navigation
- 🎯 Category icons that scroll to their respective sections

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Font Awesome
